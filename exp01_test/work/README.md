# 工作目录

由于数据集涉及知识产权，而工作目录的结果基于数据集, 本文件夹下的数据集不提供下载, 请同学们获取数据集后自行运行产生。

## 文件说明

- `Output_Masks` `/`  
  预测分隔图像文件夹  
  测试集 OCT 图像预测后的结果。分割图像中，像素值为 0 代表 RNFL 区域、像素值为 80 代表 GCIPL 区域、像素值为 160 代表脉络膜层区域、像素为 255 代表其他区域

- `Output_show` `/`  
  预测结果可视化图像文件夹  
  测试集 OCT 图像根据预测结果可视化标注的结果。分割图像中，我们用不同颜色的线标明不同层的边界
